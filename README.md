# Music player

## About

A simple and intuitive music player that allows users to:

- Play their favorite songs
- Skip tracks effortlessly
- Seek to specific points in a song
- Mark songs as favorites

## Getting Started

### Installation

Install npm packages

```bash

npm  install

```

### Running the server

In development mode:

```bash

npm  run  dev

```

In production mode:

```bash

npm  run  build

```

```bash

npm  run  preview

```
